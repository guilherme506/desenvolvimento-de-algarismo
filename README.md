# Tecnicas Desenvolvimento Algaritmo(UDF)
Repositório dedicado às atividades, exercícios e projetos desenvolvidos ao longo da disciplina de **Técnicas de Desenvolvimento de Algoritmos**.

## Sobre a disciplinas
A disciplina tem como objetivo introduzir e aprofundar os fundamentos da lógica de programação e desenvolvimento de algoritmos,
utilizando boas práticas de codificação e resolução de problemas computacionais.

## Objetivos
- Compreender os fundamentos da lógica algorítmica.
- Desenvolver algoritmos eficientes para resolver problemas diversos.
- Aplicar estruturas de controle, repetição, funções e estruturas de dados básicas.
- Praticar o raciocínio lógico e a decomposição de problemas.
