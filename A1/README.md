# Tecnicas Desenvolvimento Algaritmo(UDF)
## Projeto Final
O Projeto final proposto desse repositorio foi desenvolver um Sistema de verificação de idade para entrada em eventos

O sistema funciona da seguinte maneira:

1 -entrada de dados

O programa inicia solicitando que o usuário digite:

Seu nome

Sua idade

Caso o usuário digite "sair", o sistema encerra imediatamente.

2- Lista de Eventos Disponíveis 

No inicio do programa existe uma lista comentando quais são os eventos existentes no programa 

3- verificação de idade 

No programa eu utilizei um *for* para verificação da idade com a idade minima ja pre-colocada na lista dos eventos disponiveis
Ela funciona da seguinte maneira ele faz para cada evento um if idade_do_usuario >= idade_minima_do_evento e ira dar um print para cada if aceitado 

4 - Relátorio final

- quando o usario digitar "sair", o sistema ira mostrar todas pessoas cadastradas.
Todas as pessoas cadastradas foram armezados por uma lista
