# Tecnicas Desenvolvimento Algaritmo(UDF)
## Listas
O exercicio proprosto deste repositório foi criar um programa que receba nomes de alunos e armazene em uma lista, permitindo exibir todos os nomes ao final.

O programa funciona da seguinte maneira:

  passo 1: O usuário fala quantos alunos ele quer cadastrar
  
  passo 2: Um for é rodado de acordo com quantos alunos o usuário quer cadastrar
  
  passo 3: Cada nome digitado é colocado na lista com append()
  
  passo 4: No final, o programa lista todos os nomes armazenados
  
### Exemplo:

Quantos alunos deseja cadastrar? 4

  - Digite o nome do aluno 1: Gabriel
  - Digite o nome do aluno 2: Guilherme 
  - Digite o nome do aluno 3: Sofia
  - Digite o nome do aluno 4: Elisa

Listas de alunos cadastrados:
  - Gabriel
  - Guilherme
  - Sofia
  - Elisa
