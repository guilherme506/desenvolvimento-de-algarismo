## Estruturas_de_Repetição
O exercicio proposto deste repositório foi de desenvolver um contador de 1 a 100 com for e outro com while, exibindo apenas números pares.

O programa com *for* :

A função *for* irá percorrer uma sequência automática em um alcance de 1 ao 101 representado por *(range(1, 101))*

Em cada número, o programa testa:
    *if num % 2 == 0* Se cada numero divido por 2 for igual a zero
    ele irá dar *print num* toda vez que a condição if for aceita
